  <files>docker-compose.yml</files>
  <action>
Add a `redis_cache` service to docker-compose.yml with these exact settings:

```yaml
redis_cache:
  image: redis:7.2-alpine
  restart: always
  command: >
    redis-server
    --maxmemory 1536mb
    --maxmemory-policy allkeys-lru
    --appendonly yes
    --appendfsync everysec
  ports:
    - '6379:6379'
  deploy:
    resources:
      limits:
        memory: 1.5g
  networks:
    - career_shared_net
  volumes:
    - redis_data:/data
  healthcheck:
    test: ["CMD", "redis-cli", "ping"]
    interval: 10s
    timeout: 3s
    retries: 3
    start_period: 5s
```

Add `redis_data:` to the `volumes:` section (alongside existing `mysql_data:`).

Update the `api` service:
- Change `depends_on` from simple list to health-check form:
  ```yaml
  depends_on:
    mysql_db_api:
      condition: service_started

      condition: service_healthy
  ```
- Reduce JVM args: change `-Xms2g -Xmx4g` to `-Xms1g -Xmx2g`
- Reduce memory limit from `4g` to `3g`

Reduce `mysql_db_api` memory limit from `3g` to `2g`.

Memory budget after changes: MySQL 2GB + API 3GB + Redis 1.5GB = 6.5GB (fits 8GB server with headroom for OS).

**pom.xml changes:**

Add two dependencies AFTER the existing `spring-boot-starter-actuator` dependency (do NOT specify version — BOM-managed):

```xml

    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-redis</artifactId>
</dependency>

    <groupId>org.apache.commons</groupId>
    <artifactId>commons-pool2</artifactId>
</dependency>
```

CRITICAL: Do NOT add explicit version tags. Spring Boot 2.5.5 parent BOM manages compatible versions automatically.

**application.yml changes:**

For EACH of the three profiles (dev, staging, production), add `redis:` block AFTER the existing `cache:` block under `spring:`. Also add `management:` block at the same level as `spring:`.

**Dev profile** — add under `spring:` (after cache block):
```yaml
  redis:
    host: localhost
    port: 6379
    timeout: 2000ms
    lettuce:
      pool:
        max-active: 8
        max-idle: 8
        min-idle: 2
        max-wait: 5000ms
      shutdown-timeout: 200ms
```

And add at same indentation level as `spring:`:
```yaml
management:
  endpoints:
    web:
      exposure:

  endpoint:

      show-components: always

    redis:
      enabled: true
```
