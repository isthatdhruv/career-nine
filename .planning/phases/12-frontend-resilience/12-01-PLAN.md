  <name>Task 1: Create assessmentApi axios module with session headers, retry, and error utility</name>
  <files>react-social/src/app/pages/StudentLogin/API/assessmentApi.ts</files>
  <action>
Create a NEW file `react-social/src/app/pages/StudentLogin/API/assessmentApi.ts` with:

1. **Axios instance** (`assessmentApi`):
   - `baseURL`: `process.env.REACT_APP_API_URL || 'http://localhost:8091'`
   - Default headers: `Accept: application/json`, `Content-Type: application/json`

2. **Request interceptor** — inject session headers on every request:
   - Read `assessmentSessionToken` from `sessionStorage` -> set `X-Assessment-Session` header
   - Read `userStudentId` from `localStorage` -> set `X-Assessment-Student-Id` header
   - Read `assessmentId` from `localStorage` -> set `X-Assessment-Id` header
   - If any value is null/undefined, do NOT set that header (backwards compatible)

3. **Response interceptor** — retry with exponential backoff:
   - Track retry count on the config object: `config.__retryCount` (direct mutation, NOT cloning — axios 0.26.x requirement per research)
   - MAX_RETRIES = 3, BASE_DELAY_MS = 1000
   - Retryable conditions: `!error.response` (network error) OR `error.response.status >= 500`
   - NOT retryable: any 4xx status (400, 403, 404, 409)
   - Delay formula: `BASE_DELAY_MS * Math.pow(2, retryCount - 1)` giving 1s, 2s, 4s
   - On retry: `await new Promise(resolve => setTimeout(resolve, delay))` then `return assessmentApi(config)`
   - After max retries exhausted: `return Promise.reject(error)`

4. **Export `getErrorMessage` function** — maps AxiosError to user-friendly string:
   - No response (network error): "Unable to connect to the server. Please check your internet connection and try again."
   - 403: "Your assessment session has expired. Please return to the assessment list and start again."
   - 409: "Your assessment has already been submitted successfully."
   - 400: Use `error.response.data` if it's a string, otherwise "Invalid submission data. Please try again."
   - 500: "The server encountered an error. Your answers are saved. Please try again in a moment."
   - Default: "An unexpected error occurred. Please try again."

5. **Export** `assessmentApi` as default export, `getErrorMessage` as named export.

Do NOT import or use axios-retry. Use only built-in axios interceptor API.
