Key Design Decisions

Counsellor ≠ Faculty

Standalone entity with optional User link. Supports both internal staff and external professionals with onboarding flow.

Slot ↔ Appointment: 1:1

Each slot can have at most one appointment. Slot tracks availability state, appointment tracks the booking lifecycle.

Template → Slot Materialization

Daily cron job generates slots for next 4 weeks from active templates. template_id tracks origin; manually created slots have it null.

Generic Notification Table

Reusable for in-app bell + email triggers. reference_id + reference_type allow linking to any entity (appointment, slot, etc.).
